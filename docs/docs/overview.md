# JobCompass

JobCompass - платформа для управления поиском работы. Она помогает кандидату загрузить резюме, получить персональную ленту вакансий, сохранять интересные варианты, вести CRM по откликам, оставлять комментарии и отслеживать прогресс.

## Основные разделы

- Главная: краткое описание платформы и подсказки по работе.
- Вакансии: персональная лента вакансий с процентом совпадения.
- Сохранённые: вакансии, к которым пользователь хочет вернуться позже.
- CRM: текущие отклики, статусы, архив и аналитика.
- Профиль: резюме, навыки, технологии и настройки пользователя.

## Текущая модель работы

Разработка и тестирование выполняются локально на ПК. GitHub Pages используется для публикации статического Flutter Web и документации. Backend и база данных остаются локальными до отдельного этапа внешнего backend-хостинга.

## Главные технологии

- Frontend: Flutter Web.
- Backend: FastAPI.
- База данных: PostgreSQL.
- Миграции: Alembic.
- Документация: Docusaurus.
- Публикация frontend и docs: GitHub Pages.
