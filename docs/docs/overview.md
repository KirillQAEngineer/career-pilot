# JobCompass

JobCompass - платформа для управления поиском работы. Кандидат может заполнить профиль вручную, загрузить резюме или совместить оба варианта, а затем получать персональную ленту вакансий, сохранять интересные варианты и отслеживать отклики.

## Основные разделы

- Главная: краткое описание платформы и подсказки по работе.
- Вакансии: персональная лента вакансий с процентом совпадения.
- Сохранённые: вакансии, к которым пользователь хочет вернуться позже.
- Аналитика: текущие отклики, статусы, архив и исторические показатели.
- Профиль: ручные поля, резюме, навыки, технологии и настройки пользователя.
- Админ: статистика зарегистрированных пользователей, карточки аккаунтов и управление правами. Раздел виден только администраторам.

## Текущая модель работы

Разработка и тестирование выполняются локально на ПК. GitHub Pages публикует Flutter Web и документацию, Render запускает FastAPI backend, а Supabase предоставляет PostgreSQL.

## Главные технологии

- Frontend: Flutter Web.
- Backend: FastAPI.
- База данных: PostgreSQL.
- Миграции: Alembic.
- Документация: Docusaurus.
- Публикация frontend и docs: GitHub Pages.
